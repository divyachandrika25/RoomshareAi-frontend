# 🌐 RoomShare Web - High-Fidelity Dashboard

This is the front-facing web application for the **RoomShare** ecosystem. Built with **React**, **Vite**, and **Tailwind CSS**, it serves as a powerful dashboard for managing listings, exploring co-living spaces, and interacting with the neural matching engine.

## 🛠️ Features
- **Dynamic Property Discovery**: Sleek, high-performance interface for exploring room shares and local hotels.
- **Integrated AI Agent**: Real-time natural language search interface (e.g., "hotels in Bangalore under 5k").
- **Identity & Profiles**: Modern user profile management with behavioral tag analysis.
- **Glassmorphic Design**: Futuristic, high-end aesthetics with premium transitions.

## 🚀 Quick Start (Development)

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Run locally**:
    ```bash
    npm run dev
    ```
3.  **Build for production**:
    ```bash
    npm run build
    ```

## 🏗️ Technical Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS & Framer Motion (for animations)
- **Icons**: Lucide Icons
- **Components**: Radix UI (base for shadcn/ui components)

## 🌐 Deployment (Vercel/Netlify)

1.  Connect your GitHub repository to **Vercel** or **Netlify**.
2.  Set the Root Directory to `web/`.
3.  Ensure the build command is `npm run build` and the output directory is `dist/`.
4.  Add any necessary environment variables for API communication.

# Roomshare AI Rental Web Application

Welcome to the premium web equivalent of the Roomshare AI Rental App. This single-page application is built using modern web technologies to deliver an incredibly smooth, visually stunning experience featuring high-quality glassmorphism, fluid Framer Motion transitions, and beautiful CSS 3D live spin animations as requested.

## Tech Stack
- **Node.js**: Underlying JavaScript runtime for the development server and package management.
- **Vite & React 18**: Ultra-fast frontend framework and bundler for a seamless Single Page Application experience.
- **Tailwind CSS v4**: For the gorgeous dark mode user interface, rich gradients, and glassmorphic panels.
- **Framer Motion**: Powering the advanced page reveals, micro-animations, layout transitions, and hover effects.
- **Lucide React**: For beautiful, crisp SVG iconography.

## How to Run the Website

Follow these steps to run the website entirely locally using Node.js:

1. **Prerequisites**
   - Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
   
2. **Access the Project Directory**
   Open your terminal/command prompt and navigate to the website directory:
   ```bash
   cd c:\Users\hp\AndroidStudioProjects\roomshareAIrentalapp\website\roomshare-web
   ```

3. **Install Dependencies**
   Run the following command to install all necessary packages (React, Tailwind, Framer Motion, etc.):
   ```bash
   npm install
   ```

4. **Start the Development Server**
   Start the Vite frontend development server:
   ```bash
   npm run dev
   ```

5. **View the Website**
   Once the server starts, it will output a local URL (typically `http://localhost:5173/`).
   Open that URL in your Chrome, Edge, Safari, or Firefox browser to experience the website!
